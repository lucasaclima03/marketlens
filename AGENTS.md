# MarketLens

REST API for Brazilian retail price search, wrapping the SEFAZ Alagoas NFC-e public dataset. Includes normalization, full price history, and cross-establishment comparison.

## Agent skills

### Issue tracker

GitHub issues at `lucasaclima03/marketlens` via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`) mapped 1:1 to GitHub. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context. Read `CONTEXT.md` and `docs/adr/` at the repo root before exploring. See `docs/agents/domain.md`.
